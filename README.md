# Carbon Emission Predictor

This is a **Flask-based web application** that predicts CO₂ emissions for selected countries based on the year using a **Linear Regression model**.  

**Features:**
- Predict CO₂ emission for India, USA, China, Germany, Brazil
- Responsive web UI using **Bootstrap**
- User-friendly form input for year and country
- Shows predictions instantly on the web page
- Fully deployable using **Render**

**Tech Stack:**
- Python (Flask, Pandas, Numpy, Scikit-learn)
- HTML/CSS + Bootstrap
- Linear Regression for prediction
- Deployment on Render

**Dataset Source:**  
Dataset obtained from [Kaggle – Carbon Dioxide Emissions](https://www.kaggle.com/)

**Usage:**
1. Clone the repo  
2. Install requirements: `pip install -r requirements.txt`  
3. Run Flask app: `python app.py`  
4. Open browser: `http://127.0.0.1:5000`
